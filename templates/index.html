CNF/DNF Generator

This tool generates Conjunctive normal form(CNF), Disjunctive normal form(DNF) for propositional logic formulas. You have to follow the formats below or it will generate unexpected results.

For example, the propositional formula:
(p ∧ q) → ¬r should be written as (p and q) -> ~r
(p ⇔ q) ∨ ¬r should be written as (p <-> q) or ~r
Parenthesis must be added properly:
p -> q -> r should be written as p -> (q -> r)
